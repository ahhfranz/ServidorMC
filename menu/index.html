CONSIDERACIONES¶

  • A fecha de 11/02/2025, el servidor continúa en fase BETA.

  • Habrán muchos bugs, falta de contenido y múltiples reseteos del mismo hasta alcanzar la versión oficial 1.0

  • En caso de encontrar algún error, por favor, contacta inmediatamente con un Administrador para hacerlo saber. ¡Todos los días hay actualizaciones y soporte constante!

    • Más abajo, se explicará detalladamente todos los requisitos y procedimientos para poder entrar a jugar con nosotros.

INSTALACIÓN¶

  • Lee antes de hacer, así no cometerás errores.

    • Los pasos de instalación están enfocados en jugadores que tengan una PC promedio, con un Sistema Operativo Windows 10/11 de 64bits.

    • NO BRINDAMOS SOPORTE TÉCNICO PARA SISTEMAS OPERATIVOS Y/O ARQUITECTURAS INFERIORES.

    •  

  • 1) Descargar e instalar JAVA Temurin™ 21 LTS x64.

    • Es una dependencia tanto de Minecraft como del Launcher, sin JAVA no vas a poder ejecutar ningún ejecutable..

  • 2) Descargar e instalar SKLauncher.

    • (Opcional) En caso de tener una copia original de Minecraft, puedes saltearte este paso.

    • (Opcional) Puedes registrarte en la página de SKLauncher, y configurar un skin que los demás jugadores podrán ver.

  • 3) Una vez dentro del launcher, iniciar sesión con el nombre de usuario que elegiste en la página de SKLauncher y crear un perfil de Minecraft para la versión 1.20.1

    • (Opcional) Una vez creado, pueden editar ese perfil para modificar la cantidad de memoria RAM que quieren asignarle al juego, recomendado: 8-16GB

  • 4) Ejecutar el juego, ir a Multijugador > Añadir Servidor

    • (Opcional) El nombre del servidor puede ser elegido a gusto.

    • IP del servidor (Actualizada 10/02/2025): 201.179.143.139

PACK DE MODS (AUTORIZADOS)¶

  • El siguiente pack de mods está autorizado y aprobado para ser utilizado dentro del servidor.

  • Enlace: DESCARGAR PACK DE MODS

  • Listado de mods:

    • Xaero's Minimap

    • Xaero's World Map

    • AppleSkin

    • Optifine

Nota

  • Para poder instalar los mods, es necesario que también instales el cliente FORGE para la versión 1.20.1

    • En caso de estar usando SKLauncher, puedes crear un perfil nuevo con FORGE ya incluído, pero VERIFICÁ siempre que estés descargando la versión 1.20.1

    • Una vez instalado, ejecuta el juego una vez y cerralo al ver el menu principal, luego ve a la carpeta raíz del juego localizada en \AppData\Roaming\.minecraft, y coloca todos los archivos .jar descargados en la carpeta mods
